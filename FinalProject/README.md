Projekt aplikacji służącej do planowania wycieczek.

Lista funkcjonalności: 
- rejestracja użytkownika
- logowanie/wylogowanie użytkownika
- dodawanie planu
- dodawanie państwa do planu
- dodawanie miasta do państwa
- dodawanie miejsca do miasta
- edycja danych planu
- edycja danych Państwa
- edycja danych Miasta
- edycja danych miejsca
- usuwanie planu
- usuwanie państwa
- usuwanie miasta
- usuwanie miejsca
- wyświetlanie nazwy planu, kosztu całego planu, daty wyjazdu, liczby dni do wyjazdu dla planu który jest najbliżej aktualnej daty
- wyświetlanie listy planów w kolejności od najbliższego aktualnej dadty
- wyświetlanie szczegółów planu (nazwa, data od do, koszt całego planu, Państwo, miasto i miejsce)
- wyświetlanie szczegółów miasta (nazwa, termin, koszt dojazdu, koszt dziennego wyżywienia, koszt jednego noclegu, całkowity koszt pobytu w mieście)
- wyświetlanie szczegółów miejsca (nazwa, opis, godziny otwarcia, koszt biletu, adres www)
